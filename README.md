# spam-message-detector
The Spam Message Detector uses Machine Learning to classify messages as spam or not. Built with scikit-learn, it analyzes text patterns and predicts whether a message is unwanted. This helps users filter spam efficiently and demonstrates real-world application of AI/ML in communication systems.
# Spam Message Detector

## 📌 Description
This project detects whether a message is spam or not using Machine Learning.

## 🧠 Model
- Algorithm: Naive Bayes
- Library: scikit-learn

## ⚙️ Features
- Classifies messages as spam or ham
- Simple and fast prediction

## 🚀 How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run:
   python model.py

3. Enter message to test

## 📊 Example
Input: "Win money now"
Output: spam
